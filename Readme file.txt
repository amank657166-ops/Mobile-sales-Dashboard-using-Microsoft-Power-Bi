# Mobile Sales Dashboard

## Overview
This project is a **Mobile Sales Dashboard** built using **Microsoft Power BI**.  
The dashboard provides valuable insights into mobile sales performance through interactive visualizations and filters.

It helps users analyze:
- Sales trends
- City-wise performance
- Top mobile models
- Payment methods
- Customer ratings

All insights are displayed in a single interactive dashboard.

---

## Features
- Interactive dashboard created in **Power BI**
- Monthly sales filtering (January to December)
- City-level filtering for detailed analysis
- Displays B2B payments and cash transactions
- Dynamic revenue calculations based on filters
- Clear visualization of:
  - Total sales
  - Revenue
  - Payment types
  - Customer ratings
  - Top-performing mobile models

---

## Tech Stack
- **Microsoft Power BI**
- **Microsoft Excel** (Dataset Source)

---

## Dataset
The dataset used in this project was taken from an **Excel file** and imported into Power BI for data visualization and analysis.

---

## Installation & Usage
1. Download the project files.
2. Open the `.pbix` file in **Microsoft Power BI Desktop**.
3. Connect the Excel dataset file if required.
4. Refresh the data.
5. Explore the dashboard using filters and charts.
